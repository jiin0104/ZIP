## 개요
* 프로젝트 이름: 집탈출
* 기간: 2023.06.12-2023.07.20(약 5주)
* 인원: frontend 2명, backend 3명
* 목적: 코로나 이후 급증한 여행 심리에 따라 여행의 중요한 요소인 숙박 시설의 최신 정보를 제공하고, 예약까지 이어질 수 있는 여행 예약 플랫폼을 기획했습니다.
* 기여: frontend, Design
  * 웹 퍼블리싱(디자인), 내 정보, 예약 페이지, 예약 숙소 정보 모달창 구현, 회원 탈퇴
